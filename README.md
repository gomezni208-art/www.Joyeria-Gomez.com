# Comandos basicos de git
- 1. git status: se utiliza para saber el estado actual del repositorio
- 2. git add .: se utiliza para agregar los cambios de archivos realizados al stage
- 3. git commit -m "<cambios realizados>": se utiliza para guardar los cambios realizados en el proyecto
- 4. git push: se realiza para subir los cambios al repositorio remoto en este caso github
- 5. git branch -v: este comando ayuda a ver las ramas del repositorio
- 6. git origin -v: este comando se utiliza para ver los origenes en si los repositorios remotos
- 7. git pull: se utiliza para bajar los cambios de un repositorio
- 7. git fetch: se utiliza para saber si hay cambios en el repositorio remoto
- 8. aca te dejo un recurso para que aprendas mejor https://github.com/git-guides
